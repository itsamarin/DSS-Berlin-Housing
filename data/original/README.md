# Original Dataset

This folder contains the raw, unprocessed data collected for the Berlin Housing DSS.

## Dataset Information

- **Source**: Kaggle Berlin Rental Market Dataset
- **Total Records**: 2,319 accommodation listings
- **Data Period**: 2023-2024
- **Format**: CSV files

## Files

- `accommodations_raw.csv` - Raw accommodation listings data
- `tenants_raw.csv` - Raw tenant profiles and booking history

## Data Fields

### Accommodations
- Property ID
- Location (District, Address)
- Room Type (Entire Place, Private Room, Shared Room)
- Price per month
- Capacity
- Amenities
- Host information
- Reviews and ratings

### Tenants
- Tenant ID
- Booking history
- Preferences
- Verification status
- Geographic location

## Notes

- This data has NOT been cleaned or processed
- Contains duplicates, missing values, and inconsistencies
- Use the cleaned dataset in `/data/cleaned` for analysis
